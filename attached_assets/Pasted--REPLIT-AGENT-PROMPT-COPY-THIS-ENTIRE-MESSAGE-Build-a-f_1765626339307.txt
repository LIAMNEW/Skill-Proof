## ğŸš€ BACKEND (server.js)

Create an Express server with these endpoints:

### POST /api/analyze-github
**Input**: `{ "username": "torvalds" }`

**Logic**:
1. Fetch user data from `https://api.github.com/users/{username}`
2. Fetch repos from `https://api.github.com/users/{username}/repos?sort=updated&per_page=30`
3. Calculate language distribution from repos
4. Call Claude API with this prompt:

```javascript
const prompt = `Analyze this GitHub profile and extract technical skills.

User: ${userData.name}
Repos: ${userData.public_repos}
Followers: ${userData.followers}

Top Repositories:
${JSON.stringify(topRepos, null, 2)}

Languages:
${languageStats}

Return ONLY valid JSON:
{
  "skills": ["skill1", "skill2"],
  "proficiency_levels": {"skill1": "expert"},
  "strengths": ["strength1", "strength2"],
  "experience_summary": "brief summary",
  "notable_projects": ["project1", "project2"]
}`;
```

5. Parse Claude response and return combined GitHub + AI data

### POST /api/match-job
**Input**: `{ "profile": {...}, "jobDescription": "..." }`

**Logic**:
1. Call Claude API with:

```javascript
const prompt = `Compare candidate skills to job requirements.

Candidate Skills: ${JSON.stringify(profile.skills)}
Job Description: ${jobDescription}

Return ONLY valid JSON:
{
  "match_score": 85,
  "matching_skills": ["skill1"],
  "missing_skills": ["skill2"],
  "strengths_for_role": ["strength1"],
  "recommendation": "interview",
  "reasoning": "explanation"
}`;
```

2. Return match analysis

## ğŸ”§ TECHNICAL REQUIREMENTS

### GitHub API:
- Use fetch() with no auth (60 requests/hour is fine)
- Parse language data from repo.language field
- Sum repo sizes by language for percentages

### Claude API:
```javascript
const response = await fetch('https://api.anthropic.com/v1/messages', {
  method: 'POST',
  headers: {
    'x-api-key': process.env.ANTHROPIC_API_KEY,
    'anthropic-version': '2023-06-01',
    'content-type': 'application/json'
  },
  body: JSON.stringify({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 2000,
    messages: [{ role: 'user', content: prompt }]
  })
});
```

### Response Parsing:
- Extract JSON from Claude response (might be wrapped in markdown)
- Handle parsing errors with fallbacks
- Always return valid data structure
